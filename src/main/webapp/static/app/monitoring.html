Métriques (Metrics)

Compteurs énènementiels (Meters)

Une métrique de type Meter mesure le taux d'occurence d'un évènement par seconde, sur plusieurs intervalles donnés (cf. en-tête des colonnes)

name count mean_rate m1_rate m5_rate m15_rate

Compteurs temps (Timers)

name count mean_rate m1_rate m5_rate m15_rate mean min max p50 p75 p99

Compteurs simples (Counters)

Un compteur est une simple valeur entière (sur 64-bit) qui peut être incrémentée ou décrémentée.

name count

Valeurs ponctuelles (Gauges)

Le type de métrique Gauge est le type le plus simple et se contente de retourner une valeur. Il peut être utile par exemple pour évaluer la taille d'une collection à un moment donné.

name count

Histogrammes (Histograms)

Un Histogram mesure la distribution des valeurs pour un flux de données et permet de suivre l'évolution d'une mesure au fil du temps.

name count mean min max p50 p75 p99
